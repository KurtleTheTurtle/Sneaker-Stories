What's The Purpose Of This Site?

The purpose of this site is to provide easy access to a legitimate source of shoes. Through this site we link all of the products using StockX, the most well known site for buying legitimate sneakers.

How Can I Buy These Shoes?

If you find a shoe that you are intrested in on this site. You will be presented with a link to StockX, one of the most reliable sneaker resell marketplaces in the internet.

What If I Can't Find A Shoe On The Site?

If you cannot find a shoe through StockX, it is probably a very rare or hard to find shoe. if this is so you can submit a request to have a shoe added!
